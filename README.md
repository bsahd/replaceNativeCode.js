## 置き換えたメソッド/プロパティ

- `Math.abs()`
- `Math.max()`
- `Math.min()`
- `Math.pow()`
- `Math.exp()`
- `Math.random()` - 独自のXORShift128+実装です。1.2msくらいかかるはず
- `Math.PI`
- `Math.E`
